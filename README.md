# Late Rent Prediction System

## 📌 Problem
Late rent payments create cash flow and operational issues for property managers.

## 🎯 Objective
Identified patterns in tenant payment behavior and flagged high-risk tenants.

## 🛠 Tools Used
- SQL
- Power BI (DAX)
- Excel

## 📊 Key Insights
- Cheque payments had higher late rates
- Lower income-to-rent ratio increased risk
- Weak credit bands correlated with delays

## 📁 Files
- Dataset (CSV)
- SQL queries
- Power BI dashboard
